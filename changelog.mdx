## How to Stay Updated

Follow these steps to integrate changelog checks into your workflow.

<Steps>
  <Step title="Monitor Releases" icon="eye">
    Visit the GitHub repository regularly or set up webhooks.
  </Step>
  <Step title="Automate Checks" icon="bot">
    Use this example to fetch updates programmatically.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      async function checkChangelog() {
        const response = await fetch('https://api.github.com/repos/esen/docs/releases');
        const releases = await response.json();
        console.log('Latest:', releases[0].tag_name);
      }
      checkChangelog();
      ```
      ```python
      import requests
      response = requests.get('https://api.github.com/repos/esen/docs/releases')
      releases = response.json()
      print('Latest:', releases[0]['tag_name'])
      ```
    </CodeGroup>
  </Step>
  <Step title="Report Issues" icon="alert-triangle">
    Open a GitHub issue with reproduction steps.
  </Step>
</Steps>
